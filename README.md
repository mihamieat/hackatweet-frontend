# Hackatweet frontend
This is jour Hackatweet app frontend.
## Install 🧪
Install with
```sh
npm install yarn
yarn install
yarn build
```
## Run the frontend 🚀
```sh
yarn start
```
## Contribute 👩🏻‍🔬
- If you have not done it yet:
```sh
npm install yarn
yarn install
```
- run the dev server after making your changes.
```sh
yarn dev
```
